Kickstarter Ethereum Clone

Dapp Built with NextJS

The Kickstarter clone is supposed to replicate the idea of Crowdfunding but in a decentralized way. Instead of Kickstarter taking a 5% fee for each Campaign, the crowdfunding process is entirely peer to peer.

In this application, you can create a new crowdfunding campaign for whatever project you are looking to raise money for. You must set a minimum contribution for your peers to be considered "backers". In the demo, I set this to 4000 wei (the smallest unit of Ethereum). Once you are a "backer," you have the ability to vote on proposals made by the campaign manager. In the demo, the campaign manager is Account 1, and makes a request to hire a new software developer to help build the product that the campaign is raising money for. Since Account 2 and Account 3 are both backers to this campaign, the manager must receive approvals from both of them before he/she is able to spend the allocated amount for the request. In the demo, the campaign manager (Account 1) needs 0.4 ether to hire a new employee. Account 2 and Account 3 both approve this, and the campaign manager finalizes the request, which then distributes the 0.4 ether to the pre-determined "receiver" of the request funds (in this case, I made it simple and pre-determined that Account 2 would receive). You can see that once finalized, the request automatically distributes the 0.4 ether to Account 2.

You can check out the project on github.
